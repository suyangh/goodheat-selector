# GOODHEAT 产品选型工具

暖通产品智能选型工具 - 根据房屋信息自动推荐 GOODHEAT 热泵型号。

## 使用方式

直接访问：https://你的用户名.github.io/goodheat-selector/

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 功能

- 房屋信息采集（建筑类型、面积、年代、末端形式）
- 生活热水需求计算
- 气候校准（支持多城市）
- 智能产品推荐（R290/R32 热泵、泳池热泵、热水热泵）

## 技术栈

- HTML5 + Tailwind CSS
- 原生 JavaScript（无框架依赖）
- 单文件部署
